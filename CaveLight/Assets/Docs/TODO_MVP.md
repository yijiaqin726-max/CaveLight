# CaveLight MVP 任务拆分

## P0
- 手工搭建 Tilemap 洞穴地图并测试基础通关。
- 玩家移动与基本交互。
- 能源系统：能源持续消耗与简单补给逻辑。
- 玩家灯光效果与可见范围。
- 洞穴出口与关卡通关判定。
- 基本结算流程与胜利/失败反馈。

## P1
- 能源掉落物：玩家击破后获得补充能源。
- 天然能源节点：地图中的可采集能源点。
- 怪物：基础敌人出现、追踪与战斗交互。
- 随机洞穴生成：使用预设流程生成随机连通洞穴关卡。

## P2
- 商人房：进入商店场景或商人房后的交互。
- 商品系统：可购买的升级或补给物品。
- 正式美术替换：替换占位资源为成品视觉资源。
- 音效：玩家移动、攻击、收集、结算等音效效果。
